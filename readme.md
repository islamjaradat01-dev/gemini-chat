# Gemini Chat Web Application

This project is a modern AI chat web application built using **Flask (Python)** and the **Google Gemini API**. It provides a clean and interactive user interface that allows users to communicate with an AI model in real time.

## Features

- Send messages to the AI model
- Receive AI-generated responses in real time
- Clean and interactive user interface
- User-friendly layout inspired by modern AI tools
- Simple chat experience with a modern design

## Additional NLP Exploration

This project also explores core NLP concepts such as:

- Tokenization
- Encoding
- Decoding

These concepts were explored using **Hugging Face models** in **Google Colab**, providing deeper insight into how AI models process text internally.
## Notes
This project was developed as a simple AI chat system that demonstrates how a web application can connect with a large language model through an API and display responses in an interactive chat interface.
